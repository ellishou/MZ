アイテム、スキルに複数の属性を設定できます。  

## 設定方法
アイテム、スキルのメモ欄  
`<multElement:[id],[id],....>` データベースの設定属性に加えて属性ID[id]番の属性を持ちます。  
`<multElement:5>` 属性ID5番の属性を持ちます。  
`<multElement:5,6>` 属性ID5番と6番の複数属性を持ちます。
