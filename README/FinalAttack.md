ファイナルアタックを実装します。  

## 設定
特徴を有するメモ欄
<FinalAttack:1> 
戦闘不能になった時にファイナルアタック設定の１番で指定したスキルが発動します。
スキルのコストは考慮してませんのでスキルコストがある場合数値がマイナスに表示される場合があります。

## スキル設定
### ターゲットモード
設定スキルのターゲット  
設定スキルの範囲の対象。
とどめを刺したバトラー
戦闘不能にしたバトラーに対して攻撃します。（範囲が敵単体のみ）

設定できるスキルは複数指定できます。
