パーティコマンドをカスタマイズします。  

このプラグインを配置する場所により動作が異なります。  
下のほうに配置することにより全てのコマンドの表示順を設定できます。他プラグインによるコマンド追加を行う場合は「Commandkey」  
に該当する文字列を記入してください。  
他プラグインのコマンド追加をこのプラグインで設定せずに追加する場合は、それらのプラグインの上に配置してください。  
パーティコマンドリストの項目が１以下の時はパーティコマンドが省略されます。
