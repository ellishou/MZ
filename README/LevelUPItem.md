# [レベルアップアイテム、スキル](https://raw.githubusercontent.com/nuun888/MZ/master/NUUN_LevelUPItem.js)
# Ver.1.0.2

経験値を増減させるアイテムやスキルを作ることが出来ます。

## 設定方法
アイテム、スキルのメモ欄  
`<LevelUP:1>` レベルが1アップします。  
`<LevelUP:5>` レベルが5アップします。  
`<LevelUP:-1>` レベルが1ダウンします。  
`<LevelUP:-10>` レベルが10ダウンします。
