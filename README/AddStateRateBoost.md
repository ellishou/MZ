ステート付加率を増減する特徴を設定できます。  

## 設定方法
特徴を持つメモ欄  
`<AddStateRate:[rate]>` この特徴を持つ場合、ステートの付与率が[rate]%増減します。  
`<AddStateRate[stateId]:[rate]>` この特徴を持ち[stateId]のステートが付加されるとき、付与率が[rate]%増減します。  
`[rate]`:確率（百分率）　複数存在する場合は加算されます。  
`[stateId]`:ステートID
